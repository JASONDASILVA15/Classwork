Python types and operators

types

string

a sequence of characters enclosed in either single quotes ('') or double quotes (“”).

int

classified into zero, positive, or negative whole numbers with no fractional part and infinite precision,

float

a function or reusable code in the Python programming language that converts values into floating point numbers.

list

a data structure in Python that is a mutable, or changeable, ordered sequence of elements.

set

Sets are used to store multiple items in a single variable.

tuple

one of 4 built-in data types in Python used to store collections of data

dict

used to create a new dictionary or convert other iterable objects into a dictionary.

boolean

a Boolean variable can have only two values - True or False.
